Projects

Rental of Pets in Szczecin

Technologies: Java, Spring Framework, Thymeleaf, PostgreSQL, Hibernate, Liquibase, Docker, Lombok, QueryDSL, Docker, Maven

"Rental of Pets" is a web application designed to facilitate the rental of pets. It allows users to rent pets for a specified duration to see if they can live with animals. The application is built using Java with Spring Boot on the backend and Thymeleaf for server-side rendering. The site is made for people living in Szczecin and all payments are made directly in person.

I used Spring Boot Admin to monitor the application's performance, tried to make some metrics, and manage the health of the web application in real-time.

I created three Docker containers for the application: a Spring Boot Admin server, a web application, and a PostgreSQL database. I configured these containers to run in a shared network to ensure seamless communication and integration between the services.
